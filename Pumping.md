_Pumping_ is the process of **repeating*** the **middle section** of a string an arbitrary number of times.
 
 ![[Pumping Image.png]]

---
### Example:

For the [[alphabet]] $A = \{a, b\}$, the string `abaaba` can be produced.

| Times Pumped | Output         |
| ------------ | -------------- |
| 1            | `abaaaaba`     |
| 2            | `abaaaaaaba`   |
| 3            | `abaaaaaaaaba` |
| ...          | ...            |
