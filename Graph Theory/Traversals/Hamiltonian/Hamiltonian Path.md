A _[[Hamiltonian]]_ [[path]] is a [[path]] which visits each [[vertex]] exactly once.

---
### Example

> [!example]
> A _[[Hamiltonian]] [[Path]]_
> 
> ```mermaid
> graph LR 
> 	A --> B;
> 	B --> C;
> ```
>
> All [[Vertex|vertices]] are visited exactly once hence this is a _[[Hamiltonian]]_ [[graph]] traversal
