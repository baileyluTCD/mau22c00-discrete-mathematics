A _[[Hamiltonian]]_ [[Circuit]] is a [[circuit]] which visits each [[vertex]] exactly once.

> [!example]
> A _[[Hamiltonian]] [[Circuit]]_
> 
> ```mermaid
> graph LR 
> 	A --> B;
> 	B --> C;
> 	C --> A;
> ```
>
> All [[Vertex|vertices]] are visited exactly once hence this is a _[[Hamiltonian]]_ [[graph]] traversal
