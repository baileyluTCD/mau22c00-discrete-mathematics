An [[edge]] is said to be _incident_ to a [[vertex]] if it **connects to** that [[vertex]].

---
### Example:

![[Adjacent+Incident Example.png]]

| [[Vertex]] | Incident [[Edge\|Edges]] |
| ---------- | ------------------------ |
| $a$        | $ac$, $ad$               |
| $c$        | $ac$, $ce$               |
